Projeo de Natalidade para Projeo Populacional da Regio Metropolitana de Curitiba

Este documento apresenta os procedimentos realizados para obteno, tratamento, anlise e projeo dos dados de natalidade por grupos de faixa etria das mulheres na Regio Metropolitana de Curitiba. O resultado do modelo sero utilizados para projeo da populao da regio pelo mtodo de coortes, e ser utilizado para os diagnsticos e prognsticos que compem parte do processo de elaborao do seu Plano de Desenvolvimento Integrado.

O documento traz detalhamento das rotinas computacionais em linguagem Python utilizada, para garantir reproducibilidade e verificao do rigor tcnico por parte de quaisquer interessados.

Tratamento de Dados Nascimentos com Idade da Me Ignorada

A coluna "Idade da me" contava com maro de "Idade ignorada", para os casos em que os nascimentos no apresentaram registro de idade da me. Esses dados foram distribudos por faixas etrias, conforme a proporo de nascimentos com idade identificada. Em seguida, foram removidas as linhas com registro de "Idade Ignorada".

for ano in df_nascimentos.columns[1:]:

    #obtendo nmero de mortes com idade ignorada para determinado sexo em determinado ano

Modelagem da Variao Anual dos Dados de natalidade

O modelo exponencial, exposto na frmula adiante, foi aplicado aos dados. A escolha do modelo exponencial se deu pela sua capacidade de representar dinmicas que tende padres assintticos, que se adequam bem a padres de saturao em acrscimos ou decrscimos, o que especialmente importante no caso de diminuio de taxas de natalidade, para as quais no se pode esperar que que permaneam em diminuio a nvel zero.

$$ natalidade = f(ano) = \text{constante} \cdot e^{\text{coeficiente} \cdot (-ano - 1)} $$

A curva, com seus parmetros, constante e coeficiente, foi ajustada aos dados com base no biblioteca SciPy - Scientific Python[1]. Essa biblioteca ajusta os parmetros mtodo de Mnimos Quadrados No Lineares. Esse um processo que busca a otimizao dos quadrados dos resduos (quadrado da diferena entre valores preditos pelo modelo e valores reais). Nesse processo utilizado o mtodo de Levenberg-Marquardt, que o mais comumente utilizado em procedimentos para os quais no se apresentam restries aos valores a serem testados iterativamente como parmetros[2].

1. ^ https://docs.scipy.org/doc/scipy/reference/generated/scipy.optimize.curve_fit.html

2. ^ https://sites.cs.ucsb.edu/~yfwang/courses/cs290i_mvg/pdf/LMA.pdf

# supresso de mensagens de alerta
np.seterr(over='ignore', invalid='ignore')
warnings.filterwarnings('ignore')
warnings.filterwarnings("ignore", category=RuntimeWarning)

# Funo para o modelo exponencial decrescente
def modelo_exponencial(x, constante, coeficiente):
    return constante * np.exp(coeficiente * (-x - 1))

# DataFrame de resultados
resultados = pd.DataFrame(columns=['Faixa Etria', 'Constante', 'Coeficiente'])

# Preparao de dados necessrios ao lao de iterao
df_natalidade.reset_index(inplace=True)
faixas_etarias = df_natalidade['Faixa Etria'].unique()
num_plots = len(faixas_etarias)
num_colunas = 1
num_linhas = 6

# Criaao de subplots para cada faixa etria
fig, axs = plt.subplots(num_linhas, num_colunas, figsize=(10, 15))
fig.subplots_adjust(wspace=0.5, hspace=0.5)


# lao de iterao
for i, faixa_etaria in enumerate(faixas_etarias):
    if i >= num_plots:
        break  # Sair se todos os subplots j estiverem preenchidos
    
    # Filtragem do dataFrame para a faixa etria atual
    df_filtro = df_natalidade[df_natalidade['Faixa Etria'] == faixa_etaria]
    
    # Atribuio de dados de anos e natalidade a sua variveis para o modelo
    anos = np.array(df_filtro.columns[1:], dtype=int) - 1996
    dados_natalidade = df_filtro.iloc[0, 1:].values.astype(float)
    
    # Ajuste do modelo exponencial aos dados
    try:
        params, _ = curve_fit(modelo_exponencial, anos, dados_natalidade, maxfev=10000)
        constante, coeficiente = params
    except RuntimeError:
        constante, coeficiente = np.nan, np.nan
    
    # Criao de um DataFrame temporrio com os resultados atuais
    df_temp = pd.DataFrame({'Faixa Etria': [faixa_etaria],
                            'Constante': [constante], 'Coeficiente': [coeficiente]})
    
    # Concatenao do DataFrame temporrio ao DataFrame de resultados
    resultados = pd.concat([resultados, df_temp], ignore_index=True)

    # Calculo das previses do modelo para os anos sob anlise, inclusive os futuros, para projeo
    anos_incluso_futuros = np.arange(1, 45)
    previsao = modelo_exponencial(anos_incluso_futuros, constante, coeficiente)
    
    # Plotagem os dados originais e a curva ajustada
    axs[i].scatter(anos + 1996, dados_natalidade, label='Dados de natalidade', marker='o')
    axs[i].plot(anos_incluso_futuros + 1996, previsao, label='Curva Ajustada (Exponencial)', linestyle='--', color='red')
    axs[i].set_xlabel('Ano')
    axs[i].set_ylabel('Taxa de Natalidade')
    axs[i].set_title(f'Faixa Etria: {faixa_etaria}')
    axs[i].legend()
    axs[i].grid(True)
    axs[i].set_xlim(1995, 2040)  # Definir limite do eixo x de 1996 a 2040
    axs[i].set_xticks(np.arange(1995, 2040, 5))  # Definir marcaes no eixo x a cada 5 anos

plt.tight_layout()
