Tratamento de dados de bitos com idade ignorada

Carregamento e Tratamento de Dados de Populao - 1996 a 2022

Dados Originais

Os dados de populao foram obtidos da Contagem da Populao de 1996, bem como dos Censos Demogrficos 2000, 2010, 2022, a partir do Sistema IBGE de Recuperao de Dados - SIDRA. No momento da elaborao deste estudo, o Instituto Brasileiro de Geografia e Estatstica - IBGE -, havia divulgado apenas o total de populao por municpios, sem marcao de sexo ou faixas etrias. Por essa razo, em carter preliminar, at a divulgao dessas informaes, foi feita a distribuio da populao por sexo e faixas etrias conforme a projeo populacional do Ipardes para 2022 para os municpios da RMC. Dessa forma, garantiu-se o nmero total de populao conforme o levantamento censitrio, mas uma distribuio razovel da populao por coortes de sexo e faixa etria, conforme um estudo de metodologia robusta, aplicado por aquele instituto paranaense de pesquisa.

Clculo de Taxa de Crescimento e Estimativa de Populao em Perodos Intercensitrios

Com base nos dados populacionais de 1996, 2000, 2010 e 2022, foram calculadas as taxas geomtricas de crescimento intercensitrio para se calcular a populao em cada ano desses intervalos. Optou-se por essa abordagem, pois as estimativas anuais do IBGE, com a principal finalidade de subsidir a distribuio do Fundo de Participao dos Muncpios no so revistas aps a publicao dos resultados do Censo subsquente. Por essa razo, acredita-se que a abordagem ora adotada tenha vantagens para dar base ao clculo ao posterior clculo de mortalidade por 1.000 habitantes.

Modelagem da Variao Anual dos Dados de Mortalidade

Aps testes prvios com modelos lineares, que apresentaram casos de projeo futura de mortalidade negativa, optou-se por criar um modelo exponencial. Vrios testes foram realizados com diferentes frmulas baseadas na base neperiana (nmero de Euler) elevada a valores negativos e fracionrios de mortalidade. At que se chegou frmula adiante

$$ mortalidade = f(ano) = \text{constante} \cdot e^{\text{coeficiente} \cdot (-ano - 1)} $$

A curva, com seus parmetros, constante e coeficiente, foi ajustada aos dados com base no biblioteca SciPy - Scientific Python[1]. Essa biblioteca ajusta os parmetros mtodo de Mnimos Quadrados No Lineares. Esse um processo que busca a otimizao do quadrado dos resduos (quadrado da diferena entre valores preditos pelo modelo e valores reais). Nesse processo utilizado o mtodo de Levenberg-Marquardt, que o mais comumente utilizado em procedimentos para os quais no se apresente restries aos valores a serem testados iterativamente como parmetros[2].

1. ^ https://docs.scipy.org/doc/scipy/reference/generated/scipy.optimize.curve_fit.html

2. ^ https://sites.cs.ucsb.edu/~yfwang/courses/cs290i_mvg/pdf/LMA.pdf

# Funo para o modelo exponencial decrescente
def modelo_exponencial(x, constante, coeficiente):
    return constante * np.exp(coeficiente * (-x - 1));

# DataFrame de resultados
resultados = pd.DataFrame(columns=['Sexo', 'Faixa Etria', 'Constante', 'Coeficiente'])

# Combinaes de sexo e faixa etria
df_mortalidade.reset_index(inplace=True)
sexos = df_mortalidade['Sexo'].unique()
faixas_etarias = df_mortalidade['Faixa Etria'].unique()
num_plots = len(sexos) * len(faixas_etarias)
num_colunas = 4
num_linhas = (num_plots + num_colunas - 1) // num_colunas

# Criar subplots para cada combinao de sexo e faixa etria
fig, axs = plt.subplots(num_linhas, num_colunas, figsize=(15, 15))
fig.subplots_adjust(wspace=0.5, hspace=0.5)

for i, (sexo, faixa_etaria) in enumerate(product(sexos, faixas_etarias)):
    if i >= num_plots:
        break  # Sair se todos os subplots j estiverem preenchidos
    
    # Filtrar o DataFrame para a combinao atual
    df_filtro = df_mortalidade[(df_mortalidade['Sexo'] == sexo) & (df_mortalidade['Faixa Etria'] == faixa_etaria)]
    
    # Extrair os dados de anos e mortalidade
    anos = np.array(df_filtro.columns[2:], dtype=int) - 1996
    dados_mortalidade = df_filtro.iloc[0, 2:].values.astype(float)
    
    # Ajustar o modelo exponencial aos dados
    try:
        params, _ = curve_fit(modelo_exponencial, anos, dados_mortalidade, maxfev=10000);
        constante, coeficiente = params
    except RuntimeError:
        constante, coeficiente = np.nan, np.nan
    
    # Criar um DataFrame temporrio com os resultados atuais
    df_temp = pd.DataFrame({'Sexo': [sexo], 'Faixa Etria': [faixa_etaria],
                            'Constante': [constante], 'Coeficiente': [coeficiente]})
    
    # Concatenar o DataFrame temporrio ao DataFrame de resultados
    resultados = pd.concat([resultados, df_temp], ignore_index=True)

    # Calcular as previses do modelo para os anos futuros
    anos_futuros = np.arange(1, 45)
    previsao = modelo_exponencial(anos_futuros, constante, coeficiente);
    
    # Plotar os dados originais e a curva ajustada
    row, col = divmod(i, num_colunas)
    axs[row, col].scatter(anos, dados_mortalidade, label='Dados de mortalidade', marker='o')
    axs[row, col].plot(anos_futuros, previsao, label='Curva Ajustada (Exponencial)', linestyle='--', color='red')
    axs[row, col].set_xlabel('Ano')
    axs[row, col].set_ylabel('Taxa de bitos')
    axs[row, col].set_title(f'Sexo: {sexo}, Faixa Etria: {faixa_etaria}')
    axs[row, col].legend()
    axs[row, col].grid(True)

# Exibir os subplots
plt.tight_layout()
plt.show()
